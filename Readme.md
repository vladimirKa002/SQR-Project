# Usage

Install poetry dependencies:

```
poetry install
```

Run server:

```
cd backend
poetry run python main.py
```

Access api endpoints via `http://127.0.0.1:8000/docs`.


To generate sample data use:
```
cd backend
poetry run python database_populate.py
```

To run frontend:

```shell
streamlit run frontend/app.py
```
